What us metascripting?

Shcript is shell script that generate boilerplate shell script applications by metaprogramming boilerplate options. By using Shcript, you can build ready to go templates with a basic (bash) shell script skeleton with commands and arguments parser and many options. Shcript can generate a shell script to/from the cloud. Shcript is useful for prototype CLI automation, bigdata & cloud computing projects!.

Getting started: 

$ git clone https://github.com/shcript/shcript.git
$ cd shcript
$ chmod +x shcript
$ shcript your_app
$ shcript -cloud my_app
