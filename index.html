What is metascripting?

Shcript is a shell script that generate shell script applications by metaprogramming boilerplate options. By using Shcript, you can build ready to go CLI apps with a parser for subcommands, arguments and configuration files. Shcript can also, generate a shell script to/from a shell in the cloud.

Getting started: 

$ git clone https://github.com/shcript/shcript.git
$ cd shcript
$ chmod +x shcript
$ shcript your_app
$ shcript -cloud my_app

Useful for automation, bigdata and cloud computing

Shcript is useful for rapid prototype CLI automation, bigdata & cloud computing projects, since provides three basic profiles for these kind of applications.
