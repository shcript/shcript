Welcome to the Shcript project

Shcript is shell script basic-template generator based on autoproject and new_script. By using shcript, you can build on a fastest way, ready to go templates with a basic (bash) shell script skeleton, arguments, help modes. Moreover, a man page and a texinfo manual are generated from the help modifiers.

Getting started: 

$ git clone https://github.com/shcript/shcript.git
$ cd shcript
$ chmod +x shcript
$ ./shcript --help

Clone the repo, cd into shcript, chmod permission and, run ./shcript. Consider to copy shcrip to a bin path, in order to miss local execution (./). At the first time, you can run shcript with the help argument (--help) in order to summarize some ways to get started!
